일정확정 가이드

이지케어에 등록된(직접등록, 공단 가져오기) 방문일정을 최종 확정하여 본인부담금 청구와 직원급여에 반영하는 단계입니다.

일정확정이란?

  • 이지케어에 등록된 방문일정을 최종 확정(저장)하는 단계입니다.

  • 일정확정을 완료해야 본인부담금 산정, 급여명세서 작성이 가능해요.

일정확정 절차

  1. [4 일정확정] 클릭

  2. 해당 월 선택

  3. 공단 본인부담금과 이지케어 본인부담금 일치 여부 확인

  4. 공단과 일치 여부 한번 더 확인 원할 시 "일정 확정 전, 공단 청구명세서와 비교하기" 체크

    • 수급자 변경이력이 있을경우 일괄등록

    • 이지케어 ⟷ 공단 청구 본인부담금 비교내역에서 금액 확인

    • 금액 불일치 시 청구일정 재업로드 또는 확정 후 본인부담금액 직접 수정 가능

  5. 금액이 일치 시 "일정 일괄확정하기" 클릭

  6. 일정확정 완료 시

    • 본인부담금 서류 자동 생성

    • 근무시간이 급여에 반영

주의사항

  • 일정확정 후에는 수정이 어려우니 확정 전 모든 일정과 금액을 최종 확인하고 진행!

  • "일정확정 전" 또는 "공단 청구명세서와 비교하기" 기능으로 금액을 체크해 주세요.

  • 금액이 다를 경우 아래 항목을 확인하세요.

    • [3 방문일정 관리] > [청구일정 가져오기]로 공단 금액을 반영했는지

    • 월 중 본인부담률, 등급 변경 여부가 있었는지

    • 수급자 월 한도가 초과되었는지 등

자주 묻는 질문

일정확정 취소가 안돼요!

일정 확정 취소가 되지 않는 경우는 다음과 같은 상황일 수 있습니다.

재무회계 사용 기관인 경우

  • 해당 월의 급여명세서가 이미 생성 완료되었고, 원천세 신고까지 완료된 상태라면 일정 확정 취소가 제한됩니다.

  • 이 경우에는 재무회계 부서에 원천세 신고 마감 해제 요청을 하셔야 일정 확정 취소가 가능합니다.

마감 해제를 원하실 경우

이지케어 고객센터 1644-0340 → 내선 2번(재무회계)으로 연락해 주세요.
