시스템 특장점 02.

시스템 특장점을 유형, 타시스템 별로 나열한 테이블
LCMS(Longterm Care Management System) 유형 타 시스템 (A Type, B Type)
데이터의 코드화 및 수치화로 어르신의 상태를
실시간 분석 (예 : 임상관찰, 투약내역 등)
  • A,B Type 모두 서비스 항목별 횟수와 텍스트로만 입력 가능 함
서비스항목별 합리적인 입력방법으로 어르신의 상태를
실시간 분석 (예 : 소변량, 체위변경, 경관식, 수분공급 등)
  • A,B Type 모두 서비스 항목별 횟수와 텍스트로만 입력 가능 함
출력되는 모든 데이터는 엑셀파일, 한글파일, 텍스트 파일, pdf 파일, 이미지파일 등으로 변환
  • A Type은 가능하나 B Type은 불가능함
데이터의 변동사항을 여러가지의 그래프 유형
(막대그래프, 꺾은선그래프, 원형그래프 등)으로 표현
  • A,B Type 모두 그래프 분석은 없음
사용자의 접근성이 가장 용이
  • A Type은 비교적 접근성이 용이한 편이나 입력해야 하는 데이터가 많으므로 컴퓨터에 능숙한 사람만이 가능
  • B Type은 별도의 교육으로 사용 가능함
업무구분별(관리자, 요양보호사, 간호사, 물리치료사 등)로 분류가 명확 시스템의 기능 추가, 변경 및 업그레이드 서비스를 받을 수 있다.
  • A Type은 추가, 변경이 전혀 불가능 함
  • B Type은 추가, 변경이 제한적임
