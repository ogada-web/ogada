※ 크롬 로그인 이상 현상 해결방법 안내 ※
본 메시지는 귀하의 크롬 브라우저 설정이 변경된 경우 표시됩니다.
해당 현상은 해당 컴퓨터의 크롬 브라우저에 원인이 있습니다.
아래 방법으로 해결하고 브라우저 종료 후 브라우저를 재시작합니다.

1. 엔젤시스템 접속

2. 상단 주소창 좌측의 'X' 표시 클릭하여 화면 멈춤


3. 상단 주소창 좌측의 '자물쇠' 아이콘 클릭


4. 사이트 설정 클릭


5. 설정화면에서 마우스 스크롤하여 아래쪽 '안전하지 않은 콘텐츠' 항목을 '허용'으로 설정


6. 설정화면 상단에서 좌측 화살표 클릭한 후 창 닫기


7. 위와 같이 설정한 후 엔젤시스템을 다시 접속하면 정상입니다.
