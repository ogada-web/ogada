서비스 소개

장기요양기관의 필수 업무 『세무·재무회계』 더 이상 어려워하지 마시고
이제 이지케어에 맡기세요!

이지케어 재무회계 서비스는 재무회계 전문 운영 인력과 비영리 회계전문가 그리고 프로그램이
이지케어에 입력된 정보(종사자, 급여대장, 본인부담금)을 기반으로
쉽고, 편하게, 알아서 도와주는 서비스입니다.

서비스 주요 기능

제공 서비스

서비스 제공범위
  • W4C(사회복지시설정보시스템) 회계 업무

    01예산/결산서 작성 입력

    02W4C 회계 입력

    03추가경정(추경)

    04수입/지출 결의서 입력

    05급여대장 등록

    06영수증 증빙 처리

    07총계정원장, 계정별원장, 현금출납부 작성
    (매월 회계장부 PDF 전자 문서로 제공)

  • 세무 업무

    01원천세 신고 (근로소득세, 지방소득세)

    02퇴직금/퇴직세액 정산

    03근로소득간이지급명세서

    04연말정산

  • 4대보험 업무

    01근로자 취득/상실

    02일자리 안정자금

    03보수월액 변경신청

    04보수총액 신고

    • ※ 세무 및 4대보험 업무는 비영리 전문 세무 전문기관이 직접 처리합니다.

서비스 특징

이지케어 재무회계서비스가 특별한 이유
  • 모든 요양 업무이지케어 하나로

    일정/청구/급여 업무부터 세무·재무회계까지
    이지케어 하나로 모든 업무가 처리 가능합니다.

  • 재무회계 서비스전용 페이지 제공

    세무·재무회계 전용 페이지를 통해 업무 처리
    현황을 언제든지 확인 가능합니다.

  • 요양 본연의 업무만 제공

    이지케어는 재가장기요양기관 본연의 업무 외에는
    추가로 상품 판매를 권유하지 않습니다.
    (ex. 보험 판매)

  • 이중작업 불필요

    이지케어에 입력된 종사자, 급여대장, 본인부담금
    정보를 기반으로 업무를 처리하기 때문에 센터장님이
    이중으로 작업하실 필요가 없습니다.

  • 인증서 제공 불필요

    센터장님의 인증서 및 비밀번호를 제공할 필요가 없어
    개인 정보 노출 위험으로부터 안전합니다.

  • 영수증 전달 불필요

    통장 입/출금 내역 및 체크카드 영수증을 따로
    전달하지 않아도 결의서 입력 및 회계 처리가
    가능합니다.

이지케어는 비영리 전문 회계법인인 "선율회계법인"의 자문을 통해 원칙적인 회계처리를 하고 있으며
"노무법인 휴램프로"에서 4대보험, 노무 서비스를 제공 및 담당하고 있습니다.
