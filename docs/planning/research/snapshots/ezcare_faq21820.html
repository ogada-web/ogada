Q. [월간] 급여계약 통보

급여계약 통보서

개요

장기요양기관이 수급자와 체결한 계약내용 중 수급자, 계약 당사자, 계약기간, 급여종류 및 내용, 비용 등 월별 급여제공 내용을 장기요양기관이 공단에 통보하는 절차를 급여계약통보라고 하며 기관이 수급자에게 제공할 급여일정, 내용 등을 급여제공 전월 말일까지 노인장기요양보험 포털사이트에 통보하여야 합니다.

급여계약내용 일정등록  방법

  • 주의  전월 말일까지 해당월 일정등록 완료
  • 장기요양정보시스템 > 급여계약 > 급여계약내용 등록변경해지
  • 수급자 선택 > 급여계약내용 등록 또는 해당 월 선택 > 급여계약내용 등록(재가) 화면
  • 급여계약내용 등록(재가) > 년월/대상자 확인 > ‘입력’ 선택
  • 종사자검색 팝업창 생성 > 요양보호사 선택 > 가족여부, 시작/종료시간 입력 > ‘입력’ 선택
  • 종사자 선택완료 > 급여이용 일자 선택 > 총 이용횟수, 월금액, 월 한도액 확인 > 저장 및 통보
  • 특수한 유형의 급여제공(인지활동, 종일 방문 등)
    • 인지활동형 방문요양 : 종사자검색 팝업창에서 치매이수자 확인가능
    • 종일 방문요양 : 종사자검색 팝업창에서 구분코드(해당자만 나타남), 시작/종료시간 입력
    • 종일 방문요양 동시제공 : 동시제공 신규 목록 생성하여 구분 값 표시
    • 계약자 등록관리, 등록/변경, 장기요양급여 중복수급 금지

급여계약내용 등록 - 지난일정 수정

  • 급여제공 종료시간 이후 급여일정 등록 제한 (→ 지난일정 수정 입력하여 수정)
  • 지난일정 수정 > 수정사유 선택 > 수정 > 지난 일정 선택하여 일정 수정 > 저장 및 통보
  • 월금액 재산정 > 저장 및 통보
    • 지난일정을 수정한 수급자의 경우 청구 전 월금액 재산정 버튼을 클릭하여 한도초과금액 확인
    • 월금액 재산정 : 등록한 일정내역과 이용횟수, 월금액, 월 한도액 등이 맞지 않는 경우 사용

기타 참고사항

  • 급여계약내용 등록변경해지 - 확인사항 : 느낌표 표시가 있을 경우 확인하여야 할 사항이 있는 것으로 꼭 확인
  • 시간,날짜 등 변경사항 있을 경우 서비스 개시 전 수정,변경 (부득이 서비스 개시 이후는 지난일정 수정)
  • 급여제공 종료시간 이후 급여일정 등록 제한 → 지난일정 수정 선택 후 수정사유 입력하여 수정 가능
  • 장기요양기관은 급여계약 문서가 아닌 급여계약통보서를 공단으로 통보 (장기요양정보시스템 이용)

과거의 서비스 일정을 수정하고 싶은데 가능한가요?
① 입력 되어있던 일정을 삭제하는 경우: 클릭 → 수정사유 선택한 뒤 수정버튼 → 삭제하려는 일자의 해지
② 새로 과거의 일정을 추가하려는 경우: 클릭 → 수정사유 선택한 뒤 수정버튼 → 일정 추가하려는 날짜 체크
※ 수정 사유는 상황에 맞게 선택
기관에서 수급자의 자격(일반/감경/기초)을 확인하는 화면이 있나요?
급여계약>급여계약내용 등록변경해지>확인사항>자격변경내역 에서 변경 전 자격과 변경 후 자격 확인 가능 합니다. 급여계약>급여계약내용 등록변경해지 에서 자격이 표시되고, 월 중에 자격이 변경될 경우 급여계약해지란에 일자별로 자격 확인 가능합니다.
기관 전산화면에서는 월 한도 금액이 초과되지 않았다고 하는데, 공단에서는 월 한도 금액이 초과된 것으로 보인다고 합니다. 어떻게 해야 하나요?
수가 재산정을 위한 월금액 재산정 버튼을 누르지 않아서 발생하는 현상입니다. 기관 전산 화면에서 월금액 재산정, 저장 및 통보 버튼을 누르면 최소 5분 ~ 최대 30분 이내에 공단 전산에 적용됩니다. 지난 일정을 수정하거나, 타급여 중복으로 이용하는 수급자의 경우 청구 전 반드시 월금액 재산정 버튼을 클릭하여 금액을 확인하시기 바랍니다.
