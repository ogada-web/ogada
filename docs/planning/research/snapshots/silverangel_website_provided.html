홈페이지 디자인은 총 3가지 타입으로 제작 요청 시 한 가지 타입을 선택할 수 있습니다.
각 Type은 계절별 스킨형식으로 월이 바뀌면 자동으로 계절에 맞게 4가지의 다른 색상으로 변경됩니다.
(3월, 4월, 5월 - 봄 스킨 / 6월, 7월, 8월 - 여름 스킨 / 9월, 10월, 11월 - 가을 스킨 / 12월, 1월 2월 - 겨울 스킨)
원하시는 Type을 고르신 후 4가지 다른 색상의 스킨을 용도에 맞춰 변경해서도 사용이 가능합니다.

제작된 홈페이지를 대표 포털 사이트(네이버, 다음)에 자동으로 등록해드립니다.

Type A

심플하고 모던한 구성으로 최신 트렌드에 맞춘 디자인

카드형 구조의 심플한 레이아웃으로 산뜻하고 깔끔한 이미지를 줄 수 있습니다.
이해하기 쉬운 구성으로 홈페이지를 방문한 고객들에게 쉽게 내용을 전달하기 좋은 디자인입니다.
트렌드에 앞서가는 구조로 세련된 이미지의 Type A 디자인입니다.

  • 봄
  • 여름
  • 가을
  • 겨울

Type B

곡선을 살린 구성으로 클래식하고 친근한 디자인

라운딩을 살린 클래식하고 안정적인 레이아웃으로 친근하고 편안한 이미지를 줍니다.
상단 영역의 계절별로 사진 이미지가 바뀌어 요양원 홈페이지의 이미지가 보다 산뜻하게 다가갈 수 있습니다.

  • 봄
  • 여름
  • 가을
  • 겨울

Type C

일러스트가 강조된 아기자기하고 따뜻한 디자인

일러스트를 사용하여 따뜻하고 아기자기한 디자인으로 계절별로 메인 상단 전체 영역의 일러스트가 바뀌어 홈페이지가 지루하지 않게 느껴집니다.
영역별로 크게 구분된 컨텐츠는 가독성이 높습니다.

  • 봄
  • 여름
  • 가을
  • 겨울
